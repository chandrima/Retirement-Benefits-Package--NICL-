About Retirement Benefits Package


This package is developed for calculating the benefits available at the time of retirement. It is solely developed for the internal purpose of NIC Ltd., Head Office, Personnel Dept. For sake of testing purpose, the link is currently being distributed among all.

Instruction :
Select one of these three (1) Gratuity, (2) Leave Encashment, (3) Monetary Compensation from the left navigation bar. Each of these has three main options at header tab Form | Access Database | Generate Report .
  • Select "Form" for filling up the form.
  • Select "Generate Report" to get the report.
  • Select "Access Databse" to view/edit records from database.


    • Shortcomings and Future Scope :
    The database used here is a free-downloaded one from MySql open source forum, and the server used is a standalone Xampp (Apache) server. In future, the connectivity may hopefully change to relate with our company's running Oracle database, so that all possible records of existing employees can be automatically fetched from that database.
